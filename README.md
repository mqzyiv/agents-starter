This is an fork of cloudflare's Agent-Starter. It is powered by Workers AI and the model "@cf/meta/llama-3.2-3b-instruct" and "@cf/meta/llama-3.3-70b-instruct-fp8-fast". 
The tool I added scrapes reddit posts and comments for a specific city and leverages AI to extract restaurant names and sentiment towards those restaurants. The model then returns the restaurants with the most positive mentions.
